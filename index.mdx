H2O-3 is an open-source, distributed machine learning platform that makes it easy to train, tune, and deploy models at scale. It supports Python, R, Java, Scala, and a web-based Flow UI — and runs seamlessly on Hadoop, Spark, Kubernetes, and standalone clusters.

## Get up and running

<Steps>
  <Step title="Install H2O-3">
    Install the Python package from PyPI or the R package from CRAN.

    <CodeGroup>
    ```bash Python
    pip install h2o
    ```
    ```r R
    install.packages("h2o")
    ```
    </CodeGroup>
  </Step>
  <Step title="Start the H2O cluster">
    Initialize H2O — this starts a local cluster in-process.

    <CodeGroup>
    ```python Python
    import h2o
    h2o.init()
    ```
    ```r R
    library(h2o)
    h2o.init()
    ```
    </CodeGroup>
  </Step>
  <Step title="Load data and train a model">
    Import a dataset and train your first model.

    <CodeGroup>
    ```python Python
    df = h2o.import_file("https://s3.amazonaws.com/h2o-public-test-data/smalldata/iris/iris_wheader.csv")
    from h2o.estimators import H2OGradientBoostingEstimator
    model = H2OGradientBoostingEstimator()
    model.train(y="class", training_frame=df)
    ```
    ```r R
    df <- h2o.importFile("https://s3.amazonaws.com/h2o-public-test-data/smalldata/iris/iris_wheader.csv")
    model <- h2o.gbm(y = "class", training_frame = df)
    ```
    </CodeGroup>
  </Step>
  <Step title="Make predictions">
    Score new data and export your model for production.

    <CodeGroup>
    ```python Python
    predictions = model.predict(df)
    model.download_mojo(path="./")
    ```
    ```r R
    predictions <- h2o.predict(model, df)
    h2o.download_mojo(model, path = "./")
    ```
    </CodeGroup>
  </Step>
</Steps>

| [Deep Learning](/algorithms/deep-learning) | Complex patterns, multi-layer networks |
